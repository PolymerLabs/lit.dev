Custom Elements

Lit components are custom elements, so the browser treats them exactly like built-in elements. Create an instance with a simple HTML tag, with createElement, or with any HTML framework.

Scoped styles

Lit uses Shadow DOM, so your component's styles don't affect — and aren't affected by — any other styles on the page.

Reactive properties

When a change is made to a Lit property or its HTML attribute, Lit re-renders by precisely updating only the parts of the DOM affected by the change — without expensive VDOM diffing.

Powerful expressions

Lit uses standard template literals to bind data into templates, so you get the full expressivity of JavaScript at native speed.
