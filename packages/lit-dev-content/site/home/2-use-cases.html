Lit can build:

Standalone Components

Need to deliver interactive content or features that drop into sites you don't own, built on a variety of tech stacks? Web components are the perfect solution, and Lit makes them easy to build.

Design Systems

A design system helps you create experiences that are consistently on brand. But if your organization uses multiple frameworks, design system maintenance can become crushingly expensive. Use Lit to build one set of web components that works for every team.

Applications

Lit is also a great choice for building sites and apps. Use it as a standards-centric alternative to React or Vue, or add Lit components to progressively enhance your statically rendered site.
