The Right Balance

Lit carefully balances simplicity and opinion, giving you just enough features to reduce the most commmon boilerplate when developing web components, while avoiding building a parallel platform to the web platform. We lean on the features of the web platform and JavaScript for as much as possible.

This means that you can more easily apply your standard web development experience to Lit, and apply your experience with Lit to web development in general.

We value and strive for maximal interoperability and minimal lock-in. Protecting and amplifying your investment in your components and your skills.

Seamlessly Interoperable

Lit helps you build standards-based, seamlessly interoperable web components so your components will work with any framework, or without one.

Fast and light

Whether your end users are in emerging markets or Silicon Valley, they’ll appreciate that LitElement is extremely fast.

LitElement uses lit-html to define and render HTML templates. DOM updates are lightning-fast, because lit-html only re-renders the dynamic parts of your UI – no diffing required.

ballot

Delightfully declarative

Lit’s simple, familiar development model makes it easier than ever to build web components.

Lit's lit-html templating library lets you express your UI declaratively, as a function of state. You can use the full power of JavaScript in your templates.

sync

Really Reactive

Declare the reactive properties for your components and Lit takes care of automatically updating and re-rendering when the properties change.

Lit batches property changes to ensure easy to write tempaltes and speedy rendering.

code

TypeScript or JavaScript

Write your components in plain JavaScript or TypeScript. TypeScript users enjoy full type definitions for Lit libraries, type-checking and linting for lit-html templates, and convenient decorators for reactive properties and more.

JavaScript users don't need any build pipeline at all. In fact, you can write Lit components right in DevTools!

Organizational Freedom

Single-file or multi-file, you can write components how you want.

Code, templates. and styles can be all together, or separated, or even shared among components. Because everything uses standard JavaScript modules, Lit works great with bundlers or without them. No proprietary file types, and no special tools required.
